# PRD

## Purpose
This document defines the product scope, user value, and MVP priorities.

## MVP Scope
- Single Image Mode
- Project Mode
- Idea Chat
- Characters
- Asset Library
- Comic Studio
- Credits
- Subscription
- Export

## Product Principles
- Documentation-first development
- MVP-first delivery
- Stability over speed
- Backward compatibility by default
- Open for extension, closed for modification
- Architectural approval required before redesign

## Non-Goals
- Marketplace
- Collaboration
- Studio Mode
- Reading Platform
- Admin
- Analytics
- Enterprise features

## Documentation Rule
Any product scope decision must be captured here before implementation.
